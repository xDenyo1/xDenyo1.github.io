5. Web Research (Accessibility & Mobile Design)

Accessibility and mobile web design share many common principles. Both aim to improve usability and ensure that content is easy to access for all users, regardless of device or ability.

For example, using proper headings helps screen readers understand content structure while also making it easier for mobile users to scan information quickly. Similarly, providing alternative text for images improves accessibility for visually impaired users and also helps when images fail to load on mobile devices.

Responsive design is another key overlap. Designing flexible layouts ensures that content adapts to different screen sizes, which benefits both mobile users and users with accessibility needs.

Web developers can support both accessibility and mobile design by using semantic HTML, ensuring good contrast, avoiding fixed layouts, and testing websites on different devices.

In conclusion, accessibility and mobile design are closely related, and applying best practices for one often improves the other.
