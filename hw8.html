Part 2: Website Comparison

One appealing and one unappealing web page are reviewed below.

Appealing Website: Apple

URL: https://www.apple.com

Tables: The page does not appear to use tables for layout. Product information is organized with modern sections and grids.

CSS: Yes. CSS is used for page layout, typography, spacing, colors, responsive design, and animation.

Appeal: Appealing. The design has strong product images, clear spacing, and simple navigation. It also uses strong contrast for important calls to action.

Clear layout Strong images Consistent style

Unappealing Website: Yale School of Art

URL: https://www.art.yale.edu

Tables: The site is not mainly table-based, but the page can feel visually crowded because many content blocks compete for attention.

CSS: Yes. CSS is used for layout, color, background, and typography.

Appeal: Unappealing to me. The page uses many competing colors, inconsistent spacing, and a busy visual hierarchy.

Improvements: I would simplify the color palette, create clearer heading levels, add consistent spacing, and organize links into predictable navigation groups.
